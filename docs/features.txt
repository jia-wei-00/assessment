TaskFlow provides a broad set of features organized into five categories: Task Management, Collaboration, Planning, Reporting, and Integrations.

Task Management: Every task can have a title, description, assignee, due date, priority (Low, Medium, High, Urgent), status, and custom labels. Tasks support threaded comments, file attachments up to 100 MB each, subtasks up to 5 levels deep, and a full activity log. Tasks can be linked to one another as dependencies so that blocking relationships are visible on the board. Recurring tasks can be set to repeat daily, weekly, monthly, or on a custom cron schedule.

Collaboration: Each project has a shared inbox that consolidates all task updates and @mentions for that project. Team members can be given one of three roles: Viewer (read-only), Editor (full task editing), or Admin (manage members and settings). Guest access is available on Business and Enterprise plans, allowing external contractors to be added to specific projects without being charged as full seats.

Planning: TaskFlow includes a Gantt-chart timeline view that automatically respects task dependencies. Sprint planning is supported through sprint boards, where tasks can be moved into a sprint backlog with defined start and end dates. Milestones can be created to mark important delivery points on the timeline.

Reporting: Business and Enterprise plans include a reporting dashboard with widgets for velocity, burndown charts, time-logged vs. estimated time, and workload by team member. Reports can be scheduled to email stakeholders as PDF exports on a recurring basis.

Integrations: TaskFlow connects to GitHub and GitLab so that commits, pull requests, and branch status are visible directly on the associated task. The Slack integration sends configurable notifications to channels when tasks change status or become overdue. Google Drive and Dropbox files can be attached directly to tasks without downloading. The Figma integration embeds design frames within task descriptions for design review workflows.

Keyboard shortcuts are available throughout the TaskFlow web interface. Pressing ? on any page displays a cheat sheet of all available shortcuts. The most used shortcut is C to create a new task from anywhere in the app.
